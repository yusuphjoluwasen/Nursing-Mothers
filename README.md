# Nursing Mothers Support App

The Nursing Mothers Support App is a mobile application developed to support nursing mothers in the UK. It provides breastfeeding advice, connects mothers with others in the community, facilitates communication with healthcare professionals, and offers professional support from experts in the field.

## Features

- **Breastfeeding Advice**: Offers tips, guidance, and resources to help nursing mothers with breastfeeding.
- **Community Connection**: Allows mothers to connect with other mothers in their area or with similar experiences for support and camaraderie.
- **Healthcare Professional Communication**: Enables direct communication with healthcare professionals for advice, consultations, and assistance.
- **Professional Support**: Provides access to professional support from lactation consultants, nurses, and other experts in breastfeeding.

## Requirements

- iOS 16 or later
- Xcode 15 or later
- Swift and SwiftUI knowledge
- Firebase Messaging
- Firebase Authentication for authorization

## Installation

1. Clone or download the repository.
2. Open the project in Xcode.
3. Configure Firebase Messaging and Firebase Authentication with your Firebase project credentials.
4. Build and run the app on a simulator or physical device.
